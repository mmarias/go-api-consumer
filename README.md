# Introduction

Basic example of API that consume external endpoints.

# Requirements

1) Install golang >1.13.1

2) Clone repo on your $GOPATH
3) Dependencies
- go get -u "github.com/gorilla/mux"
4) go run main.go

Then, server is expose on localhost:8080/
